RÉSUMÉ DU PROJET
----------------------------------------------------------------------
OncoDx est un pipeline de Machine Learning conçu pour classifier 
automatiquement 5 types de cancers (sein, rein, côlon, poumon, prostate) 
à partir de données d'expression génique RNA-Seq (dataset TCGA).

Le dataset comprend 801 patients et 20 531 gènes. L'architecture 
retenue combine un filtrage statistique (ANOVA) et le modèle XGBoost, 
permettant d'allier une précision quasi-totale à une totale 
transparence clinique (explicabilité via SHAP).
